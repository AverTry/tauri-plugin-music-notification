## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-play`
- `allow-pause`
- `allow-pause-after`
- `allow-resume`
- `allow-stop`
- `allow-next`
- `allow-previous`
- `allow-seek`
- `allow-get-state`
- `allow-start-service`
- `allow-stop-service`
- `allow-set-volume`
- `allow-set-server`
- `allow-set-playing-queue`
- `allow-get-playback-session`
- `allow-clear-playing-queue`
- `allow-set-play-mode`
